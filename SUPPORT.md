# Support

Jeju Radar Training Simulator is a public, training-only simulator project.

It is not an operational ATC system, not a navigation source, not dispatch tooling, and not a certified simulator.

## Where to Ask

- Bugs: open a GitHub issue using the bug report template.
- Feature requests: open a GitHub issue using the feature request template.
- Data authority questions: use the data authority question template.
- Training scenario ideas: use the training scenario request template.
- Security or private-source exposure concerns: follow `SECURITY.md`.

## What This Project Can Support

- Browser-based radar training UI behavior.
- Deterministic scenario replay.
- Procedure and motion verification.
- Phraseology/parser contract checks.
- Public-safe data authority and documentation questions.

## What This Project Cannot Support

- Real-world ATC operations.
- Real-world navigation.
- Dispatch or safety-critical decisions.
- Certified simulator requirements.
- Redistribution of private SOPs, non-public training manuals, or local-only source material.

## Reporting Good Issues

Useful reports include:

- What you expected.
- What happened.
- Which scenario or command was involved.
- Which browser or environment was used.
- Whether the issue affects UI, procedure behavior, data authority, phraseology, or documentation.

Do not paste secrets, private source text, non-public aviation documents, or local file paths into public issues.
